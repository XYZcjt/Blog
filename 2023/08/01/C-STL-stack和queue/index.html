priority_queue常用操作

在 C++ 中，std::priority_queue 是一个优先级队列容器，它是基于堆（heap）的数据结构。优先级队列允许高优先级的元素在队列的前面，低优先级的元素在队列的后面。std::priority_queue 通常用于实现贪心算法等场景，其中需要始终访问最大或最小元素。

以下是 std::priority_queue 常用的操作：

  1. 构造函数：
    std::priority_queue 支持多个构造函数，其中最常见的是使用容器范围构造函数，可以从现有容器（如 std::vector）初始化优先级队列。

  2. push 和 emplace：
    push() 用于将元素添加到优先级队列，但 emplace() 允许你直接传递构造元素所需的参数，而无需先创建元素对象。这两个函数会根据优先级将元素放在正确的位置上。

  3. top：
    top() 函数返回优先级队列中的最大（或最小，取决于堆的类型）元素，而不会删除它。

  4. pop：
    pop() 函数用于删除优先级队列中的最大（或最小）元素。

  5. 自定义比较函数：
    默认情况下，std::priority_queue 是基于 operator< 构建的最大堆，也就是最大的元素在队列的前面。如果需要构建最小堆，可以使用自定义的比较函数或 lambda 表达式。

  6. 大小和清空操作：
    可以使用 size() 函数获取优先级队列中的元素数量，使用 empty() 函数判断队列是否为空。clear() 函数用于清空队列中的所有元素。
