插入元素：

这个成员函数用于将元素 value 插入到 std::map 容器中。如果容器中已经存在相同的键，则插入操作会被忽略，因为 std::map 中的键是唯一的。insert() 返回一个 std::pair 对象，其中的 first 成员是指向插入元素（或已经存在的相同键的元素）的迭代器，second 成员是一个 bool 值，表示插入是否成功（true 表示插入成功，false 表示插入失败）。

删除元素：

这个成员函数用于从 std::map 容器中删除元素。它有多个重载版本，可以根据迭代器或键进行删除操作。

  • erase(iterator position)：删除指定迭代器 position 所指向的键值对，并返回指向被删除元素后面键值对的迭代器。
  • erase(const key_type& key)：删除容器中键为 key 的键值对，并返回删除的键值对数量（0或1，因为 std::map 中键是唯一的）。
  • erase(iterator first, iterator last)：删除范围 [first, last) 内的所有键值对，并返回指向被删除元素后面键值对的迭代器。

unordered_map

std::unordered_map 是 C++ 中标准库提供的哈希表（散列表）容器，用于存储键值对。具有常数时间复杂度 O(1)（平均情况下，最坏情况下会退化到线性时间复杂度 O(n)）。

哈希表是一种根据键（key）快速查找值（value）的数据结构。在 std::unordered_map 中，每个键对应一个唯一的值，且键值对是按照键的哈希值进行组织和存储的。

std::unordered_map 的特点：

  • 快速查找：通过哈希函数将键映射到内部的存储桶（bucket），可以在平均情况下以常数时间复杂度 O(1) 查找值。
  • 插入和删除效率高：插入和删除键值对也具有常数时间复杂度 O(1)。
  • 无序性：std::unordered_map 不会维护元素的插入顺序，因此元素的顺序是无序的。

在 C++ 中，std::unordered_map 是通过 #include <unordered_map> 头文件引入的。
