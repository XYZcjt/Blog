可变数量的参数列表

char* format, ... 是一个函数参数，通常用于接受一个格式化字符串以及可变数量的参数列表。这种参数形式在 C 语言中非常常见，常用于类似于 printf、sprintf、scanf 等函数中。参数的含义如下：

  1. char* format: 这是一个指向字符数组（字符串）的指针，用于存储格式化字符串。格式化字符串可以包含普通字符和格式说明符。格式说明符以 % 开头，指示在字符串中插入参数的位置以及如何格式化这些参数。例如，%d 表示以整数形式插入参数，%f 表示以浮点数形式插入参数，等等。

  2. ...（省略号）: 这表示可变数量的参数列表，可以是任意数量、任意类型的参数。这些参数将根据格式化字符串中的格式说明符进行格式化并插入到字符串中。省略号是 C 语言中可变参数的表示方式。
