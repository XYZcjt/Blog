string构造参数

std::string提供了多个构造函数，用于创建字符串对象并进行初始化。下面是std::string的构造函数详解：

  1. 默认构造函数：

    创建一个空字符串对象，不包含任何字符。

  2. 字符数组构造函数：

    使用以空字符('\0')结尾的C风格字符串const char* s来初始化字符串对象。

  3. 字符数组和长度构造函数：

    使用字符数组const char* s中的前n个字符来初始化字符串对象。

  4. 重复字符构造函数：

    创建一个包含n个重复字符c的字符串对象。

  5. 复制构造函数：

    使用另一个字符串对象str来初始化当前字符串对象。复制构造函数用于创建一个与现有字符串完全相同的副本。

  6. 移动构造函数：

    使用移动语义，将另一个字符串对象str的内容转移到当前字符串对象中。这是一种高效的构造方式，因为它避免了数据的复制。

  7. 子串构造函数：

    使用字符串str中从位置pos开始的len个字符来初始化当前字符串对象。如果len未指定或为std::string::npos，则将从pos一直复制到字符串的末尾。

string字符串拼接

std::string的字符串拼接函数主要有以下几种形式的原型：

  1. + 运算符：

    这个运算符允许将两个std::string对象进行拼接，返回一个新的std::string对象，其中lhs是第一个字符串，rhs是第二个字符串。

  2. += 运算符：

    这个运算符允许将一个std::string对象与另一个std::string对象进行拼接，并将结果赋值给第一个字符串。这个函数会修改原有的字符串。

  3. append() 函数：

    这些函数用于将一个字符串追加到另一个字符串后面。第一个版本接受一个std::string对象作为参数，其他两个版本分别接受以空字符('\0')结尾的C风格字符串（const char* s）和字符数组的前n个字符作为参数。这些函数也会修改原有的字符串。

  4. push_back() 函数：

    这个函数用于将一个字符追加到字符串的末尾。

string查找函数

在C++中，std::string提供了多种字符串查找函数，用于在字符串中查找子串或字符。以下是常用的字符串查找函数的解析：

  1. find() 函数：

    这些函数用于在字符串中查找另一个字符串或以空字符('\0')结尾的C风格字符串。str是要查找的子串，pos是查找的起始位置。返回值是找到的子串在原始字符串中的位置索引（从0开始），如果未找到则返回std::string::npos。

  2. rfind() 函数：

    这些函数与find()类似，但是从后往前查找子串。pos表示查找的起始位置，如果设置为std::string::npos，则从字符串的末尾开始查找。

    这些函数用于将指定位置的子串替换为另一个字符串。pos是要替换的起始位置索引，len是要替换的子串的长度，str是要替换成的新子串。
