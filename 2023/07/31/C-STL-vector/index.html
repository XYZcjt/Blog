动态扩展

在C++中，vector是一个可以动态扩展的数组。当你向一个 std::vector 中添加元素时，如果该向量的当前大小不足以容纳新元素，它将自动分配更多的内存空间，将现有元素复制到新的内存位置，然后添加新元素，将原来的空间释放。这就是动态扩展的概念。

动态扩展的好处在于，它允许容器在需要时动态地增长，避免了在编程时预先确定容器大小的麻烦。这样，程序员可以更加专注于解决问题，而无需担心容器是否有足够的空间来存储数据。

注意：

  • 动态扩展并不是没有代价的。当容器进行动态扩展时，它可能会分配更大的内存块，并将原始数据复制到新的内存位置。这样的操作可能会导致一些性能开销。因此，在涉及大量数据的情况下，建议使用 reserve() 函数预留一些额外的容量，以避免频繁的动态扩展操作，从而提高效率。

vector存取数据

在 C++ 的 std::vector 中，可以使用多种方法来存取向量中的数据，包括使用索引访问元素、使用迭代器进行遍历、以及使用 at() 函数等。下面是一些常见的方法：

  1. 通过索引访问元素：
    使用 [] 运算符可以通过索引访问向量中的元素。索引从 0 开始，因此第一个元素的索引为 0，第二个元素的索引为 1，以此类推。

    注意： 当使用 [] 运算符访问向量元素时，如果索引超出向量范围，将导致未定义行为。

  2. 使用 at() 函数：
    at() 函数类似于使用 [] 运算符，但它提供了边界检查。如果索引超出向量范围，at() 函数将抛出 std::out_of_range 异常。

  3. 使用迭代器遍历元素：
    可以使用迭代器来遍历向量中的元素。迭代器提供了一种通用的方式来访问向量的元素，而不依赖于索引。

  4. front()
    front() 函数返回向量中的第一个元素的引用。如果向量为空，则行为未定义。使用 front() 函数时，应确保向量非空。

  5. back()
    back() 函数返回向量中的最后一个元素的引用。如果向量为空，则行为未定义。使用 back() 函数时，应确保向量非空。

vector预留空间

在 C++ 的 std::vector 中，可以使用 reserve() 函数来预留空间，即分配足够的内存以容纳指定数量的元素。预留空间可以提高向量操作的性能，特别是当向量需要频繁增长大小时。

需要注意的是，reserve() 函数并不改变向量的大小，只是增加了容量。如果要改变向量的大小，可以使用 resize() 函数。
